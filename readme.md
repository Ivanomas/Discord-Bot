Проект бот в discord на языке Python
Проект написан с целью создания бота в дискорд. Бот выполняет несколько задач, которые можно узнать при помощи комманды 'help'. 
Сам бот говорит только в специальном канале с названием 'bot'. При запросе он обрабатывает сообщение пользователя и выдаёт ответ согласно тому, что прописано в файле responses. 
В случае, когда ответ не предусмотрен, он говорит, что такой функции нет, а также дублирует сообщение пользователя. 
Бот ведёт логирование всех сообщений во всех чатах, выдавая всё в консоль. 
В будущем будут также добавлены новые функции, например, печать айди приглашения, когда оно создаётся. 
Все основные функции бота прописаны в help.
